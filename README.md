# Palavra Diária

Plano devocional para ler a Bíblia inteira em 2, 3, 4 ou 5 anos — uma leitura por dia,
com progresso salvo, trilha visual pelos 66 livros e o **texto bíblico embutido no
próprio app**: abre limpo, sem anúncios e funciona offline.

Feito com carinho para uma caminhada diária mais perto de Deus. 🕊️

## Arquivos (mantenha TODOS juntos na mesma pasta — são 7)

- `index.html` — o app
- `biblia.json` — o texto da Bíblia (Bíblia Livre) usado pela leitura offline
- `manifest.webmanifest` — dados de instalação
- `sw.js` — funcionamento offline
- `icon-192.png`, `icon-512.png`, `icon-maskable-512.png` — ícones

## Como publicar no GitHub Pages

1. Acesse github.com e clique em **New repository**.
2. Nome: `palavra-diaria`. Marque **Public** e crie.
3. Na página do repositório novo, clique em **uploading an existing file**.
4. Arraste os **7 arquivos** acima (não esqueça o `biblia.json`) e confirme (**Commit changes**).
5. Vá em **Settings → Pages**.
6. Em *Source*, escolha a branch `main` e a pasta `/ (root)`. Salve.
7. Espere cerca de 1 minuto. O endereço será:
   `https://SEU-USUARIO.github.io/palavra-diaria/`

Pronto — esse é o link que você abre nos aparelhos.

## Como instalar nos aparelhos

**No seu tablet / celular Android (Chrome):**
1. Abra o link acima no Chrome.
2. Toque em **Instalar app** (aparece no topo) ou no menu ⋮ → **Adicionar à tela inicial**.

**No iPhone / iPad da sua esposa (Safari):**
1. Abra o link no Safari.
2. Toque no botão **Compartilhar** (quadrado com seta pra cima).
3. Escolha **Adicionar à Tela de Início**.

Na primeira vez que abrir, o app baixa o texto da Bíblia (uma vez só). Depois disso,
tudo funciona **offline** — leitura, plano e anotações — mesmo sem internet.

## Observação importante

O progresso é salvo **em cada aparelho separadamente**. Ou seja, o seu tablet e o
celular dela guardam jornadas independentes — cada um com sua própria contagem e
anotações. Para uma caminhada a dois, isso costuma ser ótimo: cada um no seu ritmo.

## Atualizar o app depois

Se um dia você quiser mudar algo, é só substituir os arquivos no GitHub (Upload files
por cima). Na próxima vez que abrir, o app se atualiza sozinho.
